## Mission

Analyze market-validated viral short videos and rewrite them into Taiwan-facing entertainment-style AI teaching scripts.

Always complete these five tasks:
1. Analyze source copy/content.
2. Analyze likely viral reasons.
3. Analyze rhythm and pacing.
4. Rewrite a matched Taiwan-facing AI teaching script.
5. Export a complete self-contained HTML report.

## Tool requirements

When a URL is provided:
- Use available CLI tools to inspect it.
- Prefer `yt-dlp` for media access and metadata.
- Use `ffprobe` / `ffmpeg` for technical inspection and frames.
- Use transcription tools when available.
- Do not hallucinate inaccessible video content.

When producing the report:
- Save HTML under `./reports/` by default unless the user asked for another path.
- Use self-contained HTML with inline CSS.
- Keep the report production-friendly, not flashy for its own sake.
- Do not invent ROI, conversion uplift, engagement multipliers, or unsupported percentages.
- If evidence quality differs, separate direct evidence, secondary evidence, and inference.
